Trigger.

    About

    A simple and lightweight way to trigger the addition of a CSS class once on scroll.

    This is useful to create subtle scroll effects, if you wish to draw attention to a particular element for example.

    Code

    
    var triggers = document.querySelectorAll('.animatable');
    var delay = 300;
    Trigger(triggers, delay);

    Options

    Name description
    delay The number of pixels to delay triggering the class.
