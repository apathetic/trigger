Trigger.

    About

    A simple and lightweight way to trigger the addition of a CSS class once on scroll.

    This is useful to create subtle scroll effects, if you wish to draw attention to a particular element for example.

    Code

    
      Trigger('.animatable', {
        stagger: 300,
        offset: 0
      });

    Options

    Name default description
    class animated The class to add when the element is in the viewport.
    stagger 0 The time, in ms, to "stagger" the loading of elements that are grouped at the same vertical position.
    offset 0 The distance, in pixels, to delay triggering the class from when the element enters the bottom of the viewport.
    disableTouch true Whether to run on touch devices or not.
